# firebase-authentication-with-react-and-go
Firebase Authentication with React and Go

https://medium.com/better-programming/firebase-authentication-with-react-and-go-3784f91fa760

## Usage

### Backend

#### Start Docker

```
$ cd backend
$ docker-compose up
```

### Frontend

#### Install

```
$ cd frontend
$ yarn 
```

#### Start server

```
$ yarn start
```
